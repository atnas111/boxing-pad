# Aerobag (Boxing Pad)

🥊 **Live Demo:** [https://boxing-pad.vercel.app/](https://boxing-pad.vercel.app/)

An interactive, production-grade boxing training experience built with:
- **Next.js 16 (App Router)** & **React 19**
- **TypeScript** & **Tailwind CSS**
- **MediaPipe Hand Landmarker** for real-time kinematic tracking (running locally in the browser)
- **Framer Motion** for 3D pendulum physics and interactive animations
- **Web Audio API** for generative arcade punch sounds
- **Zustand** for state management

## Setup

```bash
npm install
npm run dev
```

Open `http://localhost:3000` to start training.
